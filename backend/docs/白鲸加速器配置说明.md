# 白鲸加速器配置说明

## 查看白鲸加速器的代理端口

### 方法一：在软件中查看

1. 打开白鲸加速器
2. 点击设置或配置
3. 查找"本地代理"、"HTTP代理"或"端口设置"
4. 记录 HTTP 代理端口号

### 方法二：查看软件界面

白鲸加速器通常在界面上会显示代理信息，查看：
- 主界面右下角或设置页面
- 可能显示类似：`127.0.0.1:端口号`

### 方法三：常见端口

白鲸加速器常用的代理端口：
- **7890** (最常见)
- **10809**
- **10808**
- **1080**

## 配置步骤

### 1. 确认代理端口

打开白鲸加速器，查看并记录 HTTP 代理端口号。

### 2. 配置到 application.yml

编辑 `backend/interest-tracker-server/src/main/resources/application.yml`：

```yaml
spring:
  http:
    proxy:
      host: 127.0.0.1        # 白鲸加速器本地代理地址
      port: 7890             # 替换为你的白鲸加速器代理端口
```

### 3. 重启应用

配置完成后，重启 Spring Boot 应用。

### 4. 测试

调用搜索接口测试是否能正常连接 TMDB API。

## 如果找不到代理端口

如果白鲸加速器没有显示代理端口，可以：

1. **查看软件日志**：软件运行日志中可能包含代理信息
2. **查看软件文档**：查看白鲸加速器的使用文档
3. **尝试常见端口**：依次尝试 7890、10809、10808、1080
4. **联系客服**：询问白鲸加速器的技术支持

## 测试代理是否工作

配置代理后，可以通过以下命令测试：

```bash
# 使用 curl 测试代理（将 7890 替换为你的端口）
curl -x http://127.0.0.1:7890 https://api.themoviedb.org
```

如果返回数据，说明代理配置正确。

## 注意事项

1. **确保 VPN 已连接**：白鲸加速器必须处于连接状态
2. **使用 HTTP 代理端口**：不是 SOCKS 端口
3. **重启应用**：修改配置后必须重启应用
4. **端口可能变化**：如果软件更新，端口可能会变化
