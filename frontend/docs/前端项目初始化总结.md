## 三、核心功能实现

### 3.1 路由系统

**路由配置** (`src/router/index.ts`)
- 实现路由守卫（登录验证）
- 支持懒加载
- 路由元信息配置

**路由列表：**
- `/dashboard` - 仪表板
- `/photo` - 摄影模块
- `/movie` - 影视模块
- `/music` - 音乐模块
- `/match` - 球赛模块
- `/timeline` - 时间线
- `/profile` - 个人中心
- `/login` - 登录页

### 3.2 状态管理 (Pinia)

**用户状态** (`src/store/user.ts`)
- 用户信息管理
- 登录状态
- Token管理

**主题状态** (`src/store/theme.ts`)
- 深色/浅色主题切换
- 主题持久化存储

### 3.3 HTTP请求封装 (`src/utils/request.ts`)

**功能特性：**
- Axios实例配置
- 请求拦截器：自动添加JWT Token
- 响应拦截器：统一错误处理

- 请求/响应日志记录

**API响应格式：**
```typescript
interface CommonResult<T> {
  code: number
  message: string
  data: T
}
```

### 3.4 布局系统

**主布局** (`src/layouts/MainLayout.vue`)
- Header组件（顶部导航）
- Sidebar组件（桌面端侧边栏）
- MobileNav组件（移动端底部导航）
- BackgroundEffects组件（背景特效）
- 响应式布局适配

**布局特点：**
- 桌面端：左侧固定侧边栏 + 右侧主内容区
- 移动端：顶部导航 + 底部导航栏
- 支持页面过渡动画

## 五、背景特效系统

### 5.1 特效组件 (`src/components/BackgroundEffects.vue`)

实现了**9种不同的背景特效方案**：

#### 基础特效（5种）
1. **网格背景** (`grid`) - 简洁的网格线条
2. **渐变光晕** (`gradient`) - 3个大型渐变光球 ⭐推荐
3. **波浪动画** (`waves`) - 底部波浪动画
4. **星空效果** (`stars`) - 100个闪烁星星
5. **几何图形** (`geometric`) - 4个旋转几何图形

#### 新颖特效（4种）
6. **粒子连线网络** (`network`) ⭐⭐⭐
   - 30个动态粒子节点
   - 自动连线形成网络
   - Canvas动态渲染

7. **能量波纹** (`ripple`) ⭐⭐
   - 5个能量波纹同时扩散
   - 雷达扫描效果
   - CSS动画实现

8. **流动光带** (`flow`) ⭐⭐
   - 8条霓虹光带流动
   - 赛博朋克风格
   - 随机角度和速度

9. **数字雨** (`matrix`) ⭐
   - 类似黑客帝国的数字雨
   - 使用日文假名和数字
   - Canvas动态渲染

### 5.2 特效切换

在 `MainLayout.vue` 中修改 `effect-type` 属性即可切换：

```vue
<BackgroundEffects effect-type="network" />
```

### 5.3 技术实现

- **Canvas特效：** network、matrix 使用 Canvas API 动态渲染
- **CSS动画：** 其他特效使用 CSS 动画和关键帧
- **性能优化：** requestAnimationFrame、will-change、硬件加速

## 七、问题解决记录

### 7.1 CSS错误修复

**问题1：** `border-border` 类不存在
- **解决：** 替换为 `border-gray-700`

**问题2：** 意外的 `}` 符号
- **解决：** 修复CSS结构，确保 `@layer` 块正确闭合

**问题3：** `via-white/3` 类不存在
- **解决：** 使用原生CSS `rgba()` 实现透明度

### 7.2 Vue警告修复

**问题：** 组件被响应式化警告
- **解决：** 使用 `markRaw()` 包装图标组件

### 7.3 背景特效显示问题

**问题：** 背景特效不显示
- **解决：**
  1. 调整 `z-index` 层级关系
  2. 使用 `position: fixed` 确保覆盖整个视口
  3. 降低主内容区域背景透明度
  4. 增强特效可见度（大小、透明度、发光效果）

### 7.4 图片加载问题

**问题：** 部分图片无法显示
- **解决：**
  1. 添加图片错误处理机制
  2. 实现图片加载状态管理
  3. 添加占位符显示
  4. 使用多个图片源作为备用
